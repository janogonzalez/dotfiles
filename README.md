# Dotfiles

My dotfiles, feel free to fork it and customize it to your own needs.

## Pre-requisites

You will need `ack` for a better experience.

## Install

    cd ~
    git clone git://github.com/janogonzalez/dotfiles.git ~/.dotfiles
    cd ~/.dotfiles
    git submodule update --init
    ./install.sh
