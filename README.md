# Dotfiles

My dotfiles, feel free to fork it and customize it to your own needs. Tested on
macOS Mojave .

## Pre-requisites

Install tmux and neovim.

    brew install tmux neovim

## Install

    git clone git://github.com/janogonzalez/dotfiles.git ~/.dotfiles
    cd ~/.dotfiles
    ./install.sh
